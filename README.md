## 系统环境和基本配置
* 本系统基于***springboot***框架和***bootstrap***框架  
  * 其中*jdk*版本为***1.8.0_261***  
  * *MySQL*版本为***8.0.29*** 
* 查询jdk版本的代码为：  
*`java -version`*
